## 1. FLOATING XP ANIMATION

When a user earns XP from any action, show a floating 
"+50 XP" (or whatever amount) text that animates upward 
and fades out over 1.5 seconds.

- Appears near the XP progress bar on dashboard
- Green colored text, bold font
- Floats up about 60px then fades to 0 opacity
- Works for all XP amounts (+50, +100, +200, +500)
- Use React Native Animated for this effect
- Should trigger from ALL payment entry points 
  (dashboard, debt detail, plan tab)

## 3. FIRST PAYMENT CELEBRATION

The very first time a user ever logs a payment 
in the app, show a special full screen celebration:

- Confetti animation
- Dex in celebrating state
- Message: "Amazing! You just took your first step 
  toward debt freedom! Keep going!"
- "+100 Bonus XP" pill shown
- Auto dismisses after 4 seconds or on tap
- Only ever shows ONCE per user (store flag 
  in AsyncStorage: firstPaymentCelebrated)

---

## 4. STREAK SHIELD

After a user reaches a 30 day streak they earn 
one Streak Shield:

- Show a small shield icon 🛡️ next to the streak 
  count on dashboard when shield is active
- If user misses a day and has a shield, the shield 
  is automatically used and streak is protected
- Show a notification: "Your Streak Shield saved 
  your streak! You're back on track 🛡️"
- Shield is consumed after use — must earn another 
  30 day streak to get a new one
- Store shield status in AsyncStorage

---

Below the Dex speech bubble, show a small rotating

- Changes every 24 hours

  1. "Paying $10 extra per month can save hundreds

  2. "The avalanche method saves the most money —

## 6. WEEKLY SUMMARY PUSH NOTIFICATION

Every Sunday at 6:00 PM local time send a weekly 
summary push notification:

- Message: "This week you paid $[total paid this week] 
  and moved [X] days closer to debt freedom. 
  Keep it up! 💪"
- Calculate total paid this week from payment logs
- Calculate days moved closer by comparing debt 
  free date from 7 days ago vs today
- If no payments logged this week, send instead:
  "A new week is a fresh start! Log a payment 
  this week to keep your momentum going 💪"
- User can toggle this off in Settings
- Default is ON
- Uses existing Expo Notifications setup

When a user adds their VERY FIRST debt to the app, 
show a celebration moment:

- Dex appears with celebrating animation

  Every journey starts with one step."
