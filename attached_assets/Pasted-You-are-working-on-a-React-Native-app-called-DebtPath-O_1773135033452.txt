You are working on a React Native app called DebtPath. Open the Strategy screen file 
AND the Main Dashboard screen file. There is a bug: the Tsunami (Custom Order) drag-
and-drop feature is not working. Users cannot drag debts to reorder them.

Fix this completely by doing the following:

STEP 1 — INSTALL THE DRAG LIBRARY:
Check if react-native-draggable-flatlist is already in package.json.
If NOT installed, run:
  npm install react-native-draggable-flatlist
  
Also ensure react-native-gesture-handler is installed (required dependency):
  npm install react-native-gesture-handler

Make sure GestureHandlerRootView wraps the app root (in App.js or index.js):
// App.js
import { GestureHandlerRootView } from 'react-native-gesture-handler';
<GestureHandlerRootView style={{ flex: 1 }}>
  <YourAppNavigator />
</GestureHandlerRootView>

STEP 2 — BUILD THE TSUNAMI DRAG LIST ON STRATEGY SCREEN:
On the Strategy screen, find the Tsunami card. When it is selected/expanded, 
render a draggable list of debts directly below it.

Replace any placeholder or non-functional list with this implementation:

import DraggableFlatList, { 
  ScaleDecorator 
} from 'react-native-draggable-flatlist';

// State for custom order (initialize from saved order or default debt order)
const [tsunamiOrder, setTsunamiOrder] = useState(debts);

// Render each draggable debt row
const renderDraggableItem = ({ item, drag, isActive }) => (
  <ScaleDecorator>
    <TouchableOpacity
      onLongPress={drag}
      disabled={isActive}
      style={{

        {item.name}
      </Text>
      <Text style={{ fontSize: 20, color: '#9CA3AF' }}>≡</Text>
    </TouchableOpacity>
  </ScaleDecorator>
);

// Render the DraggableFlatList inside the Tsunami expanded section:
{isTsunamiExpanded && (
  <View style={{ marginTop: 12 }}>
    <Text style={{ fontSize: 12, color: '#6B7280', marginBottom: 8 }}>
      Hold and drag to set your custom payoff order
    </Text>
    <DraggableFlatList
      data={tsunamiOrder}
      onDragEnd={({ data }) => handleTsunamiReorder(data)}
      keyExtractor={(item) => item.id.toString()}
      renderItem={renderDraggableItem}
      scrollEnabled={false}
    />
  </View>
)}

STEP 3 — SAVE THE CUSTOM ORDER:
When drag ends, save the new order to AsyncStorage so it persists:

const handleTsunamiReorder = async (reorderedDebts) => {
  // Add customOrder index to each debt
  const withOrder = reorderedDebts.map((debt, index) => ({
    ...debt,
    customOrder: index,
  }));
  
  setTsunamiOrder(withOrder);
  
  // Save to AsyncStorage
  await AsyncStorage.setItem('tsunamiOrder', JSON.stringify(withOrder));
  
  // Update global debts state so dashboard reflects new order
  updateDebtsOrder(withOrder); // Call whatever your global state update function is
};

STEP 4 — LOAD SAVED TSUNAMI ORDER ON MOUNT:
In the Strategy screen useEffect, load the saved tsunami order:

useEffect(() => {
  const loadTsunamiOrder = async () => {
    const saved = await AsyncStorage.getItem('tsunamiOrder');
    if (saved) {
      setTsunamiOrder(JSON.parse(saved));
    } else {
      setTsunamiOrder(debts); // Default to current debt list order
    }
  };
  loadTsunamiOrder();
}, []);
